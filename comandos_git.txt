git init . -> cria um repo git na pasta atual
git status -> mostra o status do trabalho
git add nome arquivo -> adiciona arquivo para ser comitado
git commit -m "mensagem" -> faz comit do que estava para ser comitado
git log -> mostra histórico de commit

## Fluxo de trabalho 

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init.)
3. edita ou cria os arquivos
4. git status
5. git add "nome do arquivo"
6. git status
7. git commit -m "mensagem"
8. git status
9. voltar passo 3
